# Java
## 什么是Java?
Minecraft Java 本身是由 Java代码 编写的，计算机是无法理解 Java代码

所以就要安装 Java，充当翻译官
`Minecraft_Java > Java代码 > Java > 指令集 > 计算机`

不仅仅是开服时需要 Java，在我们日常玩 Minecraft Java 时也需要 Java

比如说在第一次使用 PCL 启动 mc 时，他会说是否 自动安装Java?

总的来说，Java 是用来启动 Minecraft 和服务器的

## 各个 Minecraft 版本对应的 Java
| Minecraft 版本 | Java最低版本 | Java推荐版本 |
| ----------- | ---------- | ---------- |
| 1.7.10 =< | 8 | 8 |
| 1.8 ~ 1.15.2 | 8 | 11 |
| 1.16 ~ 1.16.5 | 11 | 16 |
| 1.17 ~ 1.19.4 | 13 | 17 |
| 1.20 => | 21 | 21 |

> [!NOTE]
> =< 等于小于
> => 等于大于

## 安装Java

> 或许有些面板服预装了 你所需的Java版本，可以跳过此理论

- Oracle JDK (通用型) [下载](https://www.oracle.com/cn/java/technologies/downloads/archive/)
