# Java
## 什么是Java?
Minecraft Java版本身是由Java代码编写的，计算机是无法理解代码的

所以就要安装Java，充当翻译官
`Minecraft_Java > Java代码 > Java > 指令集 > 计算机`

不仅仅是开服时需要Java，在我们日常玩 Minecraft Java 时也需要 Java。比如说在第一次使用PCL启动mc时，他会说是否自动安装Java?

总的来说，Java是用来启动Minecraft和服务器的
## 各个 Minecraft 版本对应的 Java
